# 🛡️ CODEX AI Cyber Shield

> ⚡ AI-Powered Cybersecurity Web Application for Real-Time Phishing Detection

---

## 🚀 Overview

**CODEX AI Cyber Shield** is a modern cybersecurity web application designed to detect and analyze phishing threats in real-time.  
Built with a sleek dark hacker-style UI and powered by intelligent detection logic, this platform helps users identify malicious links and stay protected online.

---

## 🎯 Features

🔍 **Real-Time URL Scanning**  
Detect phishing links instantly with smart analysis  

🧠 **AI-Based Detection Logic**  
Identifies suspicious patterns across multiple phishing types  

📊 **Interactive Dashboard**  
Modern cyber-style dashboard with threat insights  

⚡ **Live Progress Indicators**  
Futuristic neon progress bars with scanning animation  

🎯 **Risk Level Detection**  
- 🟢 Safe (0–10%)  
- 🟡 Suspicious (40–60%)  
- 🔴 Phishing (85–100%)

⌨️ **Keyboard UX Support**  
Press **Enter** to instantly scan links  

📁 **Multiple Phishing Types Covered**
- Email Phishing  
- Spear Phishing  
- Vishing  
- Smishing  
- Clone Phishing  
- Quishing  
- Angler Phishing  
- HTTPS Phishing  
- Evil Twin Attack  
- Whaling

## ⚙️ How It Works

1. Paste a suspicious URL  
2. Press **Enter** or click **Scan**  
3. System analyzes the link  
4. Displays:
   - Risk Percentage  
   - Threat Status  
   - Detection Result
